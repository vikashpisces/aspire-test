# Aspire

## Pre req
```
Node >=12
element-plus
Using Vue 3 setup
```
### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests (ToDo)
```
npm run test:unit
```
